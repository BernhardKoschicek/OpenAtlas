Description
===========

.. toctree::

CIDOC documentation: :cidoc_entity:`E62 String<e62-string>`

Used to add additional information.

* A description can be searched for in the advanced :doc:`/tools/search`.
* Almost all special characters are supported like Cyrillic characters or
  mathematical symbols for example.
* There are no formatting options (except line breaks) to keep data usable for
  different systems.
* Keep in mind that as much information as possible should be covered by
  using types for facilitating presentation on a presentation site
