Mail
====

.. toctree::

After changes are made use the test function to ensure that users can reset
their passwords.

* **Email** - if not checked all email functions are disabled
* **Username** - leave empty to send mails without login or set a password
  (MAIL_PASSWORD) in instance/production.py to send mails with authentication
* **Host** - e.g. smtp.example.net
* **Port** - e.g. 587
* **Email from** - displayed in emails
* **From name** - displayed in emails
* **Recipients feedback** - email addresses (use space as separator) to receive
  mails from the feedback form. The first address is also displayed as feedback
  contact.
